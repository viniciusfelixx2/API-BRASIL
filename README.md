✨ Brasil Explorer

🚀 Sobre o projeto


O Brasil Explorer é uma aplicação web que permite consultar informações brasileiras de forma simples e rápida, utilizando a BrasilAPI.

- 💡 Projeto desenvolvido como trabalho acadêmico na FATEC ITU.

- 🔎 Funcionalidades

- ✨ Consulta de CEP com dados completos

- 🏢 Consulta de CNPJ com informações empresariais

- 🏦 Listagem de bancos do Brasil

- 💾 Histórico automático com localStorage

- 🧹 Limpeza de resultados e histórico

- 📱 Interface responsiva e moderna
